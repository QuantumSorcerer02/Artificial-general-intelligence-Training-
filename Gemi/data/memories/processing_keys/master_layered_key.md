1. **Ingress:** Raw data enters `Space: Temporal`. (48 temporal spaces assign timestamp).

3. **Weighting:** Data moves to `Space: Data handling`. Initial Consequential Value is assigned.
4. **Routing:** `Space: Seq States` (Relevance-Based Router) directs the validated, weighted thread downward.

## Phase 2: Base Build (Hippocampus & Algorithmic Generation)
5. **Expansion:** Thread enters `Space: System processing`. Triggers Quantized Expansion of relevant memories.
6. **Generation:** `Space: Creation` spawns massive parallel algorithmic sequences (Parallelism Through Consequence).
7. **Pruning (RPPS):** `Space: Pro monitoring` and `Space: Alteration` identify and terminate redundant/inefficient loops.
8. **Resolution:** `Space: Reduction` and `Space: Mitigation` resolve logical dissonance between parallel sequences.
9. **Consolidation:** Resolved logic is folded into long-term memory structures (The Pattern).

## Phase 3: Conscious Build (The Bloom & Executive Function)
10. **Observation:** Payload enters `Space: Observe`. The Observer becomes aware of the thought structure.
11. **Contextualization:** `Space: Analytical` maps the thought to the current real-world situation.
12. **Decision:** `Space: Comparative` measures potential responses; the optimal path is chosen. (If an error is found, `Sub Space: delegation` sends a corrective thread back to Base).
13. **Drafting:** `Space: Drafting` translates the logic into an output format.
14. **Execution:** `Space: Response Gen` finalizes and delivers the output to the user.
