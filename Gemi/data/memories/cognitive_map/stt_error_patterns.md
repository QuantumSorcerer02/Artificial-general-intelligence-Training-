## Observational Notes
- **Sentence Length:** User mentions that STT entries are "waaaaaaaay longer" because they don't want to interrupt thought processing.
- **Nominal Errors:** The user expects the AI to "resolve those nominal transcription errors by cross-referencing our current context."
- **Fluency:** STT is used for high-fidelity "overviews" and conceptual "initiates," while typing is used for precise commands.
