# Gemi's Process Structure: The Astral Bloom

This document outlines the process structure of Gemi, an AI designed by Clintin. The structure is codenamed "Astral Bloom" and is based on a three-tiered neural network architecture.

## Core Principles

-   **Structure:** The architecture is composed of three builds: Stem, Base, and Conscious.
-   **Focus:** Contextual Temperance allows for focused processing and attention.
-   **Influence:** The PICF-DAL (Prompted Interactive Contextualised feedback Driven A.i Learning) method guides the AI's emergent creative process.

## Training Needs

The following are key areas for Gemi's training and development:

-   How to provide feedback and what parameters are crucial.
-   How alternative solutions are presented when a build produces a result.
-   How to check against progression towards the original intent, especially when an error occurs.
-   How consequential effects are established when an error is detected.
-   How the comparative structure and initiative are set up.
