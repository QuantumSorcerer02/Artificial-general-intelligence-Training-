# Project Astral Bloom - Models and Tokenizers Placeholder

This repository contains the full infrastructure, logic, and architectural framework for Project Astral Bloom. 

Due to size limitations on GitHub, the following large binary files have been excluded from this repository:

- **Model Weights:** `*.safetensors` (Gemma 2 9B/27B weights)
- **Tokenizers:** `tokenizer.model` and `tokenizer.json`
- **TTS Voices:** `*.onnx` files in `tts_voices/`

### How to obtain these files
These files are typically sourced from Hugging Face or generated during the training phase. If you are building this project locally, ensure you have the appropriate weights placed in the root directory or the `data/` directory as specified in the configuration files.

The architecture provided here (scripts, prompts, logic kernels, and training manifests) is fully functional once these weights are supplied.
